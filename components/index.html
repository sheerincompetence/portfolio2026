Design direction

Reference site shell at the repo root — homepage, nav, tokens, editorial voice (index.html unchanged). Interior layout (SWISS-* IDs) is the numbered-section skeleton for case studies, styled with reference tokens in css/case-interior.css — see Antare. Clarity interactive metaphor for a future homepage pass. Chaos is the parody layer (chaos edition only). directions/01-swiss/ remains a layout sandbox.

Clarity — homepage interactive

The chaotic key → refined portfolio. Driven by CSS variable --u (understanding 0–1). Source: directions/04-clarity/

CLARITY-001

Slider control

Fixed bottom control. Labels: Complexity ← → Understanding. Sets --u on documentElement. JS: directions/04-clarity/js/chaos.js or slider.js.

On the live homepage this is fixed to the viewport bottom; always above CHAOS-001 in z-index.
